Detailed Description

Polygon-based universe selection model that uses pre-generated universe files. Component ID: STUN001B (Alternative to STUN001A)

This selector reads from universe files generated by 'alaris universe generate':

  • Location: Alaris.Lean/Data/equity/usa/fundamental/coarse/{yyyyMMdd}.csv
  • Format: symbolId,ticker,close,volume,dollarVolume

Use this selector when LEAN fundamental data is not available. Run 'alaris universe generate' before backtesting to create the universe files.
