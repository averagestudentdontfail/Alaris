double Alaris.Core.Pricing.CREX001A.CalculateBlendingWeight ( double  timeToExpiry)
inline

Calculates the blending weight for smooth transition.

The weight function provides C⁰ continuity:

w(τ) = clamp((τ - ε_min) / zone_width, 0, 1)

At τ = ε_min: w = 0 (pure intrinsic) At τ = ε_min + zone: w = 1 (pure model)

Parameters
timeToExpiryTime to expiration in years.
Returns
Blending weight in [0, 1].
