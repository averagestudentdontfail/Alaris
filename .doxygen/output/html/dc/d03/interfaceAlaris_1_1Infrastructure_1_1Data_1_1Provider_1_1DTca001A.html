Detailed Description

Corporate Actions Provider Interface. Component ID: DTca001A.

Provides methods for retrieving and applying corporate actions for historical data adjustment in backtesting. This is critical for:

  • Maintaining point-in-time data integrity
  • Correctly aligning option strikes with underlying prices
  • Computing accurate IV term structure across events

Key Principle: Adjustment is a computation, not stored data. Raw prices are ground truth; adjustments are applied on-demand.

Live Trading Note: In live trading, IBKR handles corporate actions automatically (adjusts positions, issues new symbols). This interface is primarily for backtesting use.

Parameters

optionTradeDateThe date the option was created/traded.
actionsCorporate actions to apply.
Returns
The adjusted strike price.

When a stock splits, the OCC adjusts:

  • Strike price (divided by split ratio)
  • Contract multiplier (multiplied by split ratio)

This method handles the strike adjustment. For proper backtesting, always use this when comparing historical option strikes to adjusted equity prices.
