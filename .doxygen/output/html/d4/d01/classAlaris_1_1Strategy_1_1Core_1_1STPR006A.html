Detailed Description

Production-grade Kou model pricing using characteristic function integration. Implements semi-analytical pricing for double-exponential jump-diffusion model. The Kou (2002) model extends Black-Scholes with asymmetric jumps:

  • Uses Fourier inversion for option pricing
  • Handles fat tails and skewness
  • Calibrates to market implied volatility smile
