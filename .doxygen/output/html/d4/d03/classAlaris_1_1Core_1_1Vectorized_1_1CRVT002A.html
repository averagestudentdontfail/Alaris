Detailed Description

AVX2-accelerated transcendental functions for high-performance option pricing. Component ID: CRVT002A.

Provides hardware-intrinsic implementations of:

  • VectorExp256 - Exponential (Cody-Waite reduction + polynomial)
  • VectorLog256 - Natural logarithm (range reduction + Chebyshev)
  • VectorErf256 - Error function (Abramowitz-Stegun 7.1.26)
  • VectorNormalCDF256 - Standard normal CDF
  • VectorNormalPDF256 - Standard normal PDF

Accuracy: All functions maintain ≤ 2 ULP error for most inputs. Erf maintains ≤ 1.5e-7 max error (same as scalar CRMF001A).

Performance: 4 doubles per operation (256-bit vectors). Falls back to scalar when AVX2 is unavailable.
