Detailed Description

Unit tests for STLN001A algorithm configuration constants. Tests parameter validation without requiring LEAN runtime.

Note: STLN001A extends QCAlgorithm which requires LEAN runtime context. These tests validate configuration constants and parameter ranges only. Full algorithm testing requires LEAN backtesting environment.
