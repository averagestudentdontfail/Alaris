Detailed Description

Memory cache service for API response caching. Component ID: DTch001A.

Provides caching for expensive API calls to reduce rate limit pressure and improve performance. Implements sliding and absolute expiration policies.

Cache strategies:

  • Historical data: Long-lived (1 hour absolute)
  • Spot prices: Short-lived (5 minutes sliding)
  • Option chains: Medium-lived (15 minutes absolute)
