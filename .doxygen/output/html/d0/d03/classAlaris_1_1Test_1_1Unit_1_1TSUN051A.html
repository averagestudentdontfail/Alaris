Unit tests for FSM workflow engine. Component ID: TSUN051A.

Tests validate:

  • Determinism: Each (state, event) â†’ exactly one transition
  • Guard enforcement: Transitions blocked when guards fail
  • Action execution: Side-effects run on successful transitions
