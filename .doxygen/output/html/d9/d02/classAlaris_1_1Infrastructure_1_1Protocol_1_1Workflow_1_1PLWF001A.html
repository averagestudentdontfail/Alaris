Detailed Description

Finite State Machine engine for deterministic workflow routing. Component ID: PLWF001A.

Implements a DFA M = (Σ, S, s₀, δ, F) where:

  • Σ = Input alphabet (TEvent enum)
  • S = Finite set of states (TState enum)
  • s₀ = Initial state
  • δ: S × Σ → S = Partial transition function
  • F ⊆ S = Terminal (accepting) states

Production guarantees:

  • Determinism: Each (state, event) pair maps to exactly one transition
  • Totality: Invalid transitions return explicit errors (partial function)
  • Thread-safety: Transitions are atomic via SpinLock
  • Memory-bounded: Circular history buffer with configurable capacity
  • Immutability: FSM definition frozen after first Fire()
  • Moore actions: State entry/exit actions for clean separation

Reference: Hopcroft, Motwani, Ullman (2006) "Introduction to Automata Theory"
