Detailed Description

Exception thrown when input parameters violate algorithm bounds.

Bounds:

  • Volatility: σ ∈ [0.001, 5.0]
  • Time to expiry: τ ∈ [1/252, 30] years
  • Spot/Strike: S, K > 0
  • Moneyness: |ln(K/S)| ≤ 3
