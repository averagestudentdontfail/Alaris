Validates all inputs required for double boundary American option pricing. This is the single entry point for bounds validation before numerical algorithms.

Parameters
spotUnderlying spot price.
strikeOption strike price.
maturityTime to expiry in years.
rateRisk-free rate (can be negative).
dividendYieldContinuous dividend yield (can be negative).
volatilityImplied volatility.
Exceptions
BoundsViolationExceptionThrown if any parameter is out of bounds.

Bounds validated:

  • Spot: S > 0
  • Strike: K > 0
  • Moneyness: |ln(K/S)| ≤ 3
  • Maturity: τ ∈ [1/252, 30] years
  • Volatility: σ ∈ [0.001, 5.0]
  • Rate: finite (allows negative)
  • Dividend: finite (allows negative)

In the double boundary regime (q < r < 0), these bounds ensure the QD+ algorithm and Kim integral converge deterministically.
