◆ RateRegime

enum Alaris.Core.Pricing.RateRegime

Interest rate regime classification for American option pricing.

Enumerator
Standard 

Standard regime: r ≥ 0 or r < q < 0. Single exercise boundary. FD engine works correctly.

DoubleBoundary 

Double boundary regime: q < r < 0 for puts, or 0 < r < q for calls. Two optimal exercise boundaries exist. Requires QD+ or FP-B' method.
