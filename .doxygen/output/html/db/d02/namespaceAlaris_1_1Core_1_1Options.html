enum Alaris.Core.Options.CashDividendModel

Cash dividend model determining how discrete dividends affect pricing.

Enumerator
Spot 

Spot model: Subtract present value of dividends from current spot price. Simple, but less accurate for American options near dividend dates.

S_adjusted = S - Σ [D_i × exp(-r × t_i)] for all dividends D_i at time t_i.

Escrowed 

Escrowed model: Handle dividends as jump conditions in the FD grid. More accurate but computationally expensive.

At each dividend date, apply V(S) → V(S - D) transformation in the grid. Requires grid interpolation at dividend times.

◆ OptionType

enum Alaris.Core.Options.OptionType

Option type enumeration matching QuantLib Option.Type semantics.

Values are +1 for Call and -1 for Put to enable elegant payoff calculations:

  • Call payoff = max(S - K, 0) = max((S - K) * 1, 0)
  • Put payoff = max(K - S, 0) = max((S - K) * -1, 0) * -1
Enumerator
