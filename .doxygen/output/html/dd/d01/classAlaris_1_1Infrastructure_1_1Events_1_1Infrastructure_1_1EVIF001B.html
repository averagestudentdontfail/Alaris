Detailed Description

File-based persistent implementation of EVCR002A for production use.

Storage format: Binary (SBE-style)

  • Fixed header: 64 bytes (sequence, eventId, timestamps, lengths)
  • Variable fields: length-prefixed UTF-8 strings
  • Append-only (immutable once written)

Binary Layout per record: [0-7] SequenceNumber (long) [8-23] EventId (Guid - 16 bytes) [24-31] StoredAtUtc (long, ticks) [32-35] EventTypeLength (int) [36-39] EventDataLength (int) [40-43] AggregateIdLength (int) [44-47] AggregateTypeLength (int) [48-51] CorrelationIdLength (int) [52-55] InitiatedByLength (int) [56-59] TotalRecordLength (int) [60-63] Reserved/Checksum [64...] Variable-length string data
