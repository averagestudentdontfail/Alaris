Detailed Description

Interactive Brokers implementation of execution quote provider. Component: DTib005A | Category: Data Provider | Variant: A (Primary)

This provider supports two modes of operation:

  • Live/Paper Mode: Connects to IB Gateway for real-time quotes
  • Backtest Mode: Uses simulated quotes based on market data

Configuration is read from appsettings.jsonc InteractiveBrokers section:

"InteractiveBrokers": {
"Host": "127.0.0.1",
"Port": 4002, // 4002=paper, 4001=live
"ClientId": 1,
"TradingMode": "paper"
}
