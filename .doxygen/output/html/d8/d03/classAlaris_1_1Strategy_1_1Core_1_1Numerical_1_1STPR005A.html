Detailed Description

Production-grade Differential Evolution optimizer for global optimization. Component ID: STPR005A.

Particularly effective for non-convex, multi-modal objective functions. DE is a population-based stochastic optimization algorithm that:

  • Doesn't require gradient information
  • Handles non-smooth, non-convex objectives
  • Good at escaping local minima
  • Robust for calibrating jump-diffusion models (Heston, Kou)

Mutation Strategy: DE/rand/1/bin (standard, most widely used)

Library Status: Custom implementation maintained in-house. Neither MathNet.Numerics nor Accord.Math provide DE optimization.
