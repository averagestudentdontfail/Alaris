Detailed Description

Spectral collocation American option pricing engine.

Implements the Andersen-Lake-Offengenden algorithm which achieves ~10⁻⁸ accuracy with sub-millisecond pricing by interpolating the exercise boundary on Chebyshev nodes and using Gauss quadrature for integration.
