Detailed Description

AVX2-accelerated option chain batch pricing for high-throughput Greeks computation. Component ID: STPR010A.

Provides end-to-end batch pricing for option chains using SIMD acceleration. Processes entire chains of options for a single underlying in one call.

Target use case: Computing Greeks for all strikes/expiries in a chain for delta hedging.
