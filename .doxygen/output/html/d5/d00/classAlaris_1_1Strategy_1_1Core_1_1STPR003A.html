Detailed Description

Production-grade Heston model pricing using characteristic function integration. Implements the semi-analytical pricing formula from Heston (1993). Uses Carr-Madan Fourier inversion or Lewis (2001) approach for option pricing, then backs out implied volatility using Newton-Raphson iteration.
