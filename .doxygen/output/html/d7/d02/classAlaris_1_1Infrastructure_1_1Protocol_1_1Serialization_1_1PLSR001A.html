Detailed Description

Zero-allocation binary serialization for Alaris market data types. Component ID: PLSR001A.

This class implements SBE-style encoding with fixed layouts and direct memory access. All encode/decode operations work on Span<byte> with no heap allocations.

Binary Format (little-endian):

  • Header: 8 bytes (blockLength:2, templateId:2, schemaId:2, version:2)
  • Fixed fields: at known offsets
  • Variable-length data: at end with length prefix
