Detailed Description

Polygon.io REST API client for market data retrieval. Component ID: DTpr001A.

Implements DTpr003A (Market Data Provider interface) using Polygon.io API via Refit declarative interface (IPolygonApi).

Note: Polygon does NOT provide historical Greeks/IV via API. This provider calculates IV from historical option prices using Black-Scholes, which is the industry-standard approach for backtesting.

Resilience provided by Microsoft.Extensions.Http.Resilience standard handler.
