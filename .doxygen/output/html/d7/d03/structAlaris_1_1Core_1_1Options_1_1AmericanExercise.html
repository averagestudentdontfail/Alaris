Detailed Description

American exercise specification. Replaces QuantLib AmericanExercise.

American options can be exercised at any time between the earliest and latest exercise dates. For standard American options:

  • EarliestDate = valuation date (immediate exercise allowed)
  • LatestDate = expiration date
