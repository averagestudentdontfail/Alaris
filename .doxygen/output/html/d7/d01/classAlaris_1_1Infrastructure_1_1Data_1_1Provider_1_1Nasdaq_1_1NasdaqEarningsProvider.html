Detailed Description

NASDAQ public API earnings calendar provider. Component ID: DTea001C.

Implements DTpr004A (Earnings Calendar Provider interface) using NASDAQ's public API via Refit declarative interface (INasdaqCalendarApi).

API Endpoint: https://api.nasdaq.com/api/calendar/earnings Cost: FREE (no API key required) Rate Limit: Undocumented (appears unlimited for reasonable use)

Fail-fast design: No fallback providers. Errors propagate immediately. Resilience provided by Microsoft.Extensions.Http.Resilience standard handler.

Future Alternative: EODHD Calendar API ($19.99/month) provides 100,000 calls/day with official C# wrapper and SLA if this endpoint becomes unreliable.
