Detailed Description

Unified American option pricing engine. Automatically selects the appropriate method based on rate regime.

Rate Regime Classification (Healy 2021)

Regime Condition Method
Standard r ≥ 0 OR r < q < 0 FD (CREN002A)
Double Boundary q < r < 0 (puts) OR 0 < r < q (calls) QD+ (DBAP002A)
