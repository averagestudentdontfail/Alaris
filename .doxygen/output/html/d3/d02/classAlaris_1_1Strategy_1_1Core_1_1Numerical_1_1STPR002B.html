Detailed Description

AVX2-accelerated Gauss-Legendre integration for characteristic function pricing. Component ID: STPR002B.

Provides batch-vectorized numerical integration optimized for:

  • Heston characteristic function integration
  • Kou jump-diffusion pricing
  • General semi-infinite integrals with exponential decay

Uses 32-point Gauss-Legendre quadrature with batch evaluation of 4 points at a time. For smooth characteristic functions, this provides ~3x speedup over scalar integration.
