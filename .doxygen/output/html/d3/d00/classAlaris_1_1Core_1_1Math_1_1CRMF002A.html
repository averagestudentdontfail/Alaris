Refines a characteristic root using Super-Halley iteration.

Parameters
λInitInitial root estimate.
aQuadratic coefficient (σ²/2).
bLinear coefficient (r - q - σ²/2).
cConstant coefficient (-r).
toleranceConvergence tolerance.
maxIterationsMaximum iterations.
Returns
Refined root value.

Super-Halley iteration formula: λₙ₊₁ = λₙ - (f(λₙ)·f'(λₙ)) / (f'(λₙ)² - ½f(λₙ)·f''(λₙ))

For quadratic: f(λ) = aλ² + bλ + c f'(λ) = 2aλ + b f''(λ) = 2a
