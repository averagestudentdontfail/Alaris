Corporate Action Data Model. Component ID: DTmd003A.

Represents a single corporate action event that may require price adjustments in historical backtesting data.

Key Fields:

  • ExDate: The ex-dividend/split date (price adjusts on this date)
  • Factor: The adjustment multiplier (e.g., 2.0 for 2:1 split)
  • PayDate: When the action is paid/effective
Parameters
SymbolTicker symbol affected by this action.
ExDateEx-date when the adjustment takes effect.
TypeType of corporate action.
FactorAdjustment factor. For splits: new shares per old share. For dividends: cash amount per share.
DescriptionHuman-readable description of the action.
PayDateOptional payment/effective date.
RecordDateOptional record date.

Gets whether this action requires price adjustment.

Gets whether this action requires strike adjustment for options.

Computes the price adjustment multiplier.

Returns
The multiplier to apply to pre-action prices. For a 2:1 split, returns 0.5 (price is halved).

Computes the strike adjustment multiplier for options.

Returns
The multiplier to apply to pre-action strikes. For a 2:1 split, returns 0.5 (strikes are halved).
