Detailed Description

Polygon Universe Provider - fetches daily aggregates for all US stocks. Component ID: DTpr006A.

Uses Polygon's Grouped Daily API to efficiently retrieve OHLCV data for all US equities on a given date. This enables universe selection without requiring QuantConnect's fundamental data subscription.
